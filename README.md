# foot-switch
フットペダルスイッチ用のKarabiner-Elementsの設定

# 注意事項
使用しているフットペダルスイッチは以下の通り

* ルートアール メカニカルスイッチ搭載 USBフットペダルスイッチ 3ペダル グレー RI-FP3MG
    * vendor_id: 16701
    * product_id: 8455
    * 左ペダル: a 
    * 真ん中ペダル: b
    * 右ペダル: c

各種設定ファイル内のvendor_id,product_id, ペダルのキーバインドは上記のフットペダルのものであるため、  
別のフットペダルを使用する場合は適宜変更すること。
